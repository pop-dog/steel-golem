# Steel Golem

A personal Director assistant for the [Draw Steel](https://mcdm.gg/DrawSteel) TTRPG. Steel Golem tracks campaign state — Heroes, Villains, NPCs, Factions, Locations, Encounters, and more — and assists with session planning through structured workflows and rules reference.

It is implemented as a set of Claude Skills and a `steel-golem` CLI, backed by a Campaign Database of markdown files with YAML frontmatter.
