# Steel Golem

A personal Director assistant for the [Draw Steel](https://mcdm.gg/DrawSteel) TTRPG. Steel Golem is a campaign tracker, session planning assistant, and rules reference, implemented as a set of Claude Skills and a `steel-golem` CLI.
